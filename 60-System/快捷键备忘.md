# 快捷键备忘

## VSCode + Foam

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+P` | 命令面板 |
| `Ctrl+Shift+P` → `Foam: Open Daily Note` | 打开每日笔记 |
| `Ctrl+Shift+P` → `Foam: Show Graph` | 显示知识图谱 |
| `Ctrl+点击 [[链接]]` | 打开/创建笔记 |
| `[[` | 触发笔记链接补全 |

## Markdown

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 粗体 |
| `Ctrl+I` | 斜体 |
| `Ctrl+Shift+V` | 预览 |
| `Ctrl+K V` | 侧边预览 |
| `Ctrl+Shift+P` → `Markdown: 创建目录` | 生成 TOC |

## Claude Code

| 命令 | 功能 |
|------|------|
| `/obsidian` | 同步 Obsidian |
| `/docx` | 导出 Word |
| `/pdf` | 导出 PDF |
