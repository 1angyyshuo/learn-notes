## 🚀 转换为 PDF 的方法（推荐）

### 方法1：浏览器打印（最简单）

1. 打开 VSCode，安装插件 **"Markdown Preview Enhanced"**
2. 打开 `论文阅读报告_Quest.md`
3. 右键 → "Markdown Preview Enhanced: Open Preview"
4. 在预览页面右键 → "Open in Browser"
5. 浏览器中按 `Ctrl+P`（或 `Cmd+P`）
6. 目标打印机选择 **"另存为 PDF"**
7. 保存为：`学号_姓名_作业2.pdf`

### 方法2：使用 VSCode 插件（最方便）

1. 安装插件 **"Markdown PDF"**
2. 打开 `论文阅读报告_Quest.md`
3. 右键 → **"Markdown PDF: Export (pdf)"**
4. 自动在当前目录生成 PDF

### 方法3：使用 pandoc（需安装）

```bash
# 安装 pandoc 和 xelatex
# Windows: winget install pandoc
# Mac: brew install pandoc

# 转换
pandoc "论文阅读报告_Quest.md" -o "学号_姓名_作业2.pdf" --pdf-engine=xelatex -V CJKmainfont="SimSun"
```
